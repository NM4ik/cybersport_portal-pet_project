## Installation
install using pip
```
>For windows

$ pip install
```
install DRF
```
pip install djangorestframework
```
Add ```'rest_framework'``` to your ```INSTALLED_APPS``` setting.
```
INSTALLED_APPS = [
    ...
    'rest_framework',
]
```

## Example
Startup up a new project like soo..
```
pip install django
pip install djangorestframework
django-admin startproject example .
./manage.py migrate
./manage.py createsuperuser
```
